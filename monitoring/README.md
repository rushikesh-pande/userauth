## Log Correlation

Every HTTP request generates:
- `X-Correlation-ID` header (passed through or generated)
- `correlationId` MDC field in ALL log entries

Search in Kibana: `correlationId: "your-id"` to trace a single request
across all services.

## Adding Business Metrics

```java
@Autowired
private BusinessMetricsService metrics;

// In your service method:
metrics.incrementActive();
try {
    // ... do work ...
    metrics.recordSuccess("createOrder");
} catch (Exception ex) {
    metrics.recordFailure("createOrder", "database");
    throw ex;
} finally {
    metrics.decrementActive();
}
```
